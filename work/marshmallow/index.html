Prototype

● An APP for group to save money together fulfills different life stage.
● Users : couple, family, friends
● Example goals:
❍ Family save money to travel aborad together.
❍ Couple save money for a new car.
● Why the APP called marshmallow?
❍ A book 《Don't Eat The Marshmallow Yet!》which I have read many years ago to encourage to save it and enjoy a greater happiness. The story is from Stanford marshmallow experiment.
● About APP :
❍ Send the link to your friend or family to build a group.
❍ Create a marshmallow box, for instance, Travel box, House box etc.
❍ Create a new marshmallow goal, for exmaple, A trip to Japan require $60,000. Both couple save $2,000 every month and repay it 15 months.
The marshmallow goal will be able to unlock when it is achieved.
❍ If you want to give up before achieving the goal, you will be charge an amount in order to unlock it. (Unlock fare is set when create a goal, you can select 5%-30% of savings as the fare by yourself.)
❍ If you achieved the marshmallow goal, you will earn the marshmallow interest as a reward.

● 一款群組共同儲蓄的APP，實現人生不同階段的需求。
● 使用者：情侶、夫妻、家人、朋友
● 使用情境範例：
❍ 情侶一起存錢出國。
❍ 夫妻一起存錢買車。
● 為什麼是棉花糖？
❍ 因為小時候看的一本書「孩子，先別急著吃棉花糖」，與源自史丹佛棉花糖實驗的故事。
● 關於APP:
❍ 發送邀請連結建立群組帳戶
❍ 建立棉花糖收納盒，如旅遊、房屋等
❍ 建立一個棉花糖目標，如日本旅遊，共同目標存款$60,000元，每個人每月存$2,000元，持續15個月，達成後即可開鎖棉花糖。
❍ 如中途放棄則需要支付開鎖費用給平台(建立時自己設定5%-30%的帳戶費用)
❍ 而成功達成也可以獲得利息獎勵(於設定開鎖費同時，平台會計算相對的利息獎勵)
